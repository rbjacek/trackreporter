A complete verification package.

Track Reporter reports can include route metadata, field work type, addresses, route metrics, coverage area, map preview, photo preview, timeline, markers, photo categories and evidence pages.

Route Id Each report is linked to a specific recorded route for easier identification.
Document created The report contains a creation timestamp, useful for audit and documentation.
From / To addresses Start and end locations are converted into readable addresses.
Coverage area The report can show the area covered during route tracking.
Average / max speed Route metrics include average speed and maximum recorded speed.
Photo categories Photos can be grouped as Before Work, After Work, Evidence, Work Completed or Other.
Photo preview page The report can include a dedicated preview section before full photo evidence pages.
Evidence details Each photo evidence item can include created time, GPS coordinates, evidence type and address.
